9. (  )은 하드웨어와 소프트웨어를 클라우드 안에 숨기고, 사용자가 필요한 서비스 만을 이용하는 컴퓨팅 환경을 가리킨다.
- 클라우드 컴퓨팅

- 유닉스
16. 컴퓨터 하드웨어는 (  ), (  ), 입력장치, 출력장치, 저장장치의 다섯가지 요소로 구성된다.
- CPU, 메인 메모리
17. 특정 기계에 포함된 작은 컴퓨터를 (  )이라고 한다.

19. 키보드와 모니터가 개발되면서 작업이 진행되는 중간에 사용자에게 입력을 받거나 사용자에게 중간 결과 값을 보여줄 수 있는 시스템을 (  )이라고 한다.
- 대화형 시스템
20. 다음 중 가장 큰 단위는? TB, GB, EB, KB
- EB
21. 다음 중 속도를 나타내는 단위가 아닌 것은? Hz, GB, rpm, bps
- GB
22. 100MB 파일을 100Mbps 네트워크로 전송하면 약 몇 초가 걸리는가? bps(bits per sec) b/s(byte per sec)
- 10초
23. 10진수 25.25는 2진수로 얼마인가? -> 소수점 아래만 가져와 x2 하는 턴마다 결과값의 정수 부분(1.~ or 0.~)을 소수점 아래에 붙혀준다.(x2 했을때 결과값이 1이 넘어가면 정수 부분의 1을 없애고 완벽한 1.0 이 나오거나 이전 소수값과 같은 소수값이 나올 때까지 x2 반복.)
- 11001.01
24. 16진수 A7은 10진수로 얼마인가?
- 167
25. 10진수 17을 16진수로 변환하면 어떤 값이 되는가?
- 11
26. -15.25(10진수)을 IEEE 754 기준(32비트)로 정규화 했을 때, 지수부(exponent)만 적어보시오.
  - 10000010
  - IEEE 754 기준 정규화 하는 법
  1.양수이면 0, 음수이면 1을 32비트 -> 32칸의 첫칸에 삽입.
  2.부호를 떼어낸 숫자의 절대값을 2진수로 표현. (15.25 == 1111.01)
  3.소수점을 2진수의 가장 왼쪽에서 두번째자리로 옮긴다. (1.11101)
  4.소수점을 옮긴 만큼 2를 곱한다. (1.11101 x 2^3)
  -------------여기까지 정규화된 표현 방식------------------
  5.이후 소수점 아래 부분을 부호자리 1칸(1비트), 지수자리 8칸(8비트)제외한 뒷부분 23칸(23비트)에 넣어준다. - 남는 자리엔 0
  6.4번에서 구한 2^3에서 지수 3에 bias인 127을 더한다. (3+127 = 130)
  7.130을 2진수로 변환. (10000010)
  8.앞서 비워둔 지수자리 8칸에 넣어준다.(1 10000010 11101000000000000000000)
  9.부동소수점 방식으로 정규화 완성 - 11000001011101000000000000000000
27. 다음 중 보라색 값은? #(ff00ff) #(00ff00) #(0000ff) #(00ffff)
- #(ff00ff)
28. 2의 보수에서 -3을 나타내는 것은?
- 1101 (1의 보수 = 0010, 2의 보수 = 0011 == -3)
29. 입력 값이 모두 다를 경우에만 결과 값이 참이 되는 연산자는?
- XOR
30. 1초 동안 클록 틱이 몇 번 발생했는지를 나타내는 단위는 ( )이다.
- 헤르츠(Hz)
31. 문자를 처리하려고 숫자와 문자를 하나씩 대응하는 7비트 코드를 (   )라고 한다.
- 아스키 코드
32. 알고리즘의 실행 순서를 결정하는 제어 구조 중 특정 조건을 만족하는지를 나타낸 구조는?
- 선택 구조
33. 각 부품을 연결할 뿐 아니라 각 부품에 전기를 공급하여 원활하게 작동하도록 하는 것은?
- 버스
34. 다음 중 버퍼의 일종으로 필요할 것이라 예상되는 데이터를 미리 가져오는 것은?
- 캐시
35. CPU 같은 저장 장치에서 메인 메모리로 데이터를 가져오는 명령, 저장 장치로 데이터를 내보내는 명령을 내리는 장치는?
- 제어 장치
36. 다음 중 레이저를 이용한 저장 장치로 디스크 한 장에 50GB를 저장 할 수 있는 것은?
- 블루레이
37. 다음 중 그래픽과 사운드를 동시에 모니터나 TV로 전송할 수 있는 것은?

38. 버스 여러 개에서 데이터가 지나다니는 하나의 통로를 (  )이라고 한다.
- 채널
39. 속도 차이가 나는 두 장치 사이에서 속도 차이를 완화시켜주는 장치를 ( )라고 한다.
- 버퍼
40. CPU는 명령에 따라 계산을 하는 산술 논리 연산 장치 (ALU), 다른 장치에 명령 신호를 내이는 제어 장치, 처리한 데이터를 임시로 보관하는 (  )로 구성된다.
- 레지스터
41. 다음 중 나머지 셋과 종류가 다른 것은? OGG / JPG / PNG / BMP
- OGG(음향, 비손실 압축 파일)
42. 다음 중 동영상 파일 확장자가 아닌 것은? MOV / GIF / MP4 / AVI
- GIF
43. 다음 중 나머지 셋과 압축 방식이 다른 것은? ZIP / MOV / JPG / AVI
- ZIP(비손실 압축)
44. 다음 중 일반적인 오디오 CD의 양자화 값은 얼마인가? 44.1khz / 96khz / 32bit / 16bit
- 16bit(44.1khz로 샘플링 하고 16bit로 양자화)
45. 40khz/10bit 의 스테레오 음악 10초를 만들었다. 파일 크기는 얼마 인가?
- 40000x10x2(스테레오)
46. 가로 6000픽셀, 세로 4000픽셀의 사진이 있다. 파일 크기는 얼마인가?
- 픽셀 하나의 크기는 3Byte(R, G, B 각각 1Byte 씩).
- 6000x4000x3 = 72MB

- 샘플링
48. 아날로그 - 디지털 변환 과정에서 눈금을 사용하여 디지털 값으로 변환하는 것을 ( )라고 한다.
- 양자화
49. 운영체제가 사용자나 응용프로그램이 하드웨어에 직접 접근하지 못하게 막음 으로써 컴퓨터 자원을 보호하는 함수의 집합은?
- 시스템 호출
50. 프로세스의 주요 상태 중 프로세스 제어 블록(PCB)을 만드는 상태는?
- 생성
51. ( )는 실제 메모리의 크기와 상관없이 프로세스에 커다란 메모리 공간을 제공하는 기술이다.
52. 메모리가 모자라서 쫒겨난 프로세스는 저장 장치의 특별한 공간에 모아두는데 이 영역을 무엇이라 하는가?
- 스왑
53. 저장 공간에 저장된 파일의 위치 및 관련 정보를 모아 놓은 것은?
- 파일 테이블
54. 하드디스크 같은 저장 장치의 저장 단위는 섹터이지만, 운영체제와 저장 장치 간 데이터 전송 단위는 (  )이다.
- 블럭
55. 파일에는 파일 이름, 버전, 크기, 만든 날짜와 같은 정보가 포함된 (  )가 존재 한다.
- 헤더
56. 컴퓨팅 사고의 구성 요소 중 문제에서 중요하지 않은 부분을 제거하고 중요한 특징만으로 문제를 구성함으로써 문제 해결을 좀 더 쉽게 하는 것을 (  )라고 한다.
- 추상화
57. 요일별로 판매량을 조사해보니 일요일에 많이 팔린다는 것을 알게 되었다면 컴퓨팅 사고의 구성 요소 중 ( )을 사용한 것이다.
- 패턴인식
58. 프로그래밍 언어 중 변수와 함수를 하나로 묶어서 편리하게 처리 할 수 있게 하는 언어를 ( ) 언어라고 한다.
- 객체 지향
59. 다음 중 올바르지 않은 것은?
① 컴퓨터를 이용하여 문제를 해결하는 논리적이고 창의적인 생각 방식을 컴퓨팅 사고라고 한다.
② 컴퓨팅 사고의 네 가지 구성 요소로 분해, 패턴 인식, 추상화, 객체 지향이 있다.
③ 컴퓨팅 사고는 지넷 윙(Jeannette Wing) 박사가 기고한 논문에서 언급되었다.
④ 컴퓨팅 사고는 컴퓨터(사람이나 기계)가 효과적으로 수행할 수 있도록 문제를 정의하고, 그 답을 기술하는 것이 포함된 사고 과정 일체를 가리킨다.
- 2 (추상화, 분해, 패턴인식, 알고리즘)
60. 컴퓨팅 사고의 네 가지 구성 요소를 설명한 것 중 올바르지 않은 것?
① 컴퓨팅 사고의 네 가지 구성 요소 중 문제를 해결하기 쉬운 작은 단위로 나누는 것은 분해다. 
② 문제에서 중요하지 않은 부분을 제거하고 중요한 특징만으로 문제를 구성함으로써 문제해결을 좀 더 쉽게 하는 것이 추상화다.
③ 데이터들의 특징별로 나누어 유사한 문제 해결 방식이 있는지 찾아보는 과정은 패턴 인식이다.
④ 어떠한 문제를 해결하는 일련의 절차나 방법을 공식화한 형태로 표현한 것은 프로그램이다
- 4 (어떠한 문제를 해결하는 일련의 절차나 방법을 공식화한 형태로 표현한 것은 알고리즘 이다)
